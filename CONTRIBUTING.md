# Contributing To GearDex (Public Repo)

Thanks for helping improve GearDex.

## Scope

This repository is for public collaboration on:

- Bug reports
- Feature requests
- Release feedback

This repository is not used for open-source code contributions.

## Before Opening An Issue

- Confirm you are on the latest app version available in Releases.
- Search existing issues to avoid duplicates.
- Keep reports specific and reproducible.

## Bug Reports

Please include:

- Platform and version (macOS/Windows/Linux)
- GearDex app version
- Clear reproduction steps
- Expected behavior
- Actual behavior
- Screenshots or logs with sensitive data removed

## Feature Requests

Please include:

- The problem you are trying to solve
- Why current behavior is insufficient
- Proposed outcome

## Security Issues

Do not post security vulnerabilities publicly. See `SECURITY.md`.
