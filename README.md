This repository is the public home for **GearDex** issue tracking for both the web app and macOS app, plus official **macOS app** releases.

Website: [www.geardex.app](https://www.geardex.app)

## Purpose

Use this repository to:

- Report issues for the GearDex web app at [www.geardex.app](https://www.geardex.app)
- Report issues for the GearDex macOS app
- Request features for the web app or macOS app
- Download official macOS app releases and pre-releases

## What GearDex Does

GearDex helps photographers and videographers:

- Track camera gear inventory and value
- Log maintenance history and service details
- Manage gear loans and returns
- Store documents and notes tied to equipment
- Use Iris AI assistant features for gear and workflow help

- Open a web or macOS bug report: [Issues](https://github.com/dylandersen/geardex-public/issues/new/choose)
- Download GearDex for macOS: [Releases](https://github.com/dylandersen/geardex-public/releases)
- Contact GearDex support: [www.geardex.app/contact](https://www.geardex.app/contact) or `geardex.help@gmail.com`
- Review release notes and channels: `RELEASES.md`
- Get support guidance: `SUPPORT.md`
- Report vulnerabilities privately: `SECURITY.md`
